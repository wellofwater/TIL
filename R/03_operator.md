# 연산

> NA 값이 있을 경우 연산x, na.rm을 통해 제외해줘야한다.

### sum

1. sum

   ```R
   # 모든 값을 더해준다
   > tt <- sum(sh[,c(4:7)])
   > tt
   [1] 200036
   ```

   

2. rowSums

   1. 결측치 X

      ```R
      > tt <- rowSums(sh[,c(4:7)])
      > tt
      [1]  20016  60002 120018
      ```

   2. 결측치 O

      ```R
      # 결측치가 있을 경우, 그냥 더해주면 해당 값은 결측치가 된다.
      > tt <- rowSums(sh[,c(4:7)])
      > tt
      [1]  20016     NA 120018
      
      # na.rm=T를 통해 결측치를 연산에서 제외한다
      > tt <- rowSums(sh[,c(4:7)],na.rm = T)
      > tt
      [1]  20016  60002 120018
      ```
